-> O que é, Como e Onde usar .NET

	- O que é ?
	Infraestrutura para desenvolvimento de software criada pela
microsoft
	Uma aplicação .net é desenvolvida para e roda em uma das 
seguintes implementações do .net:
	> .NET Core
	> .NET Framework
	> Mono
	> Universal Windows Plataform(UWP)

	- Como ?
	.NET Standard é como se fosse um contrato
	Cada implementação inclui um ou mais .NET Runtimes
	(ambiente de execução):
	> .NET Core: CoreCLR e CoreRT
	> .NET Framework: CLR
	> Mono: Mono Runtime
	> UWP: .NET Native

	Autalmente a microsoft desenvolve e suporta 3 linguagens 
para .NET
	C# 	F# 	VB

	- Onde ?
	A microsoft quer estar em todo os lugares
