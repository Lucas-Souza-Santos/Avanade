-> Array 
    Um array é uma estrutura de dados que contém um número X de
    elementos, todos do mesmo tipo, que são acessados através 
    de índices computados

    Arrays são tipos de referência e a declaração de uma
    variável array simplesmente reserva espaço para uma 
    referência de uma instância de array. 

    int[,] a2 = new int[10, 5]; duas dimensões
    int[,,] a3 = new int [10, 5, 2]; três dimensões 

-> Inicializador de Array 
    int[] a = new int[] {1, 2, 3};
    int[] a = {1, 2, 3};
    int[] t = new int[3];
    t[0] = 1;
    t[2] = 2;
    t[3] = 3;
