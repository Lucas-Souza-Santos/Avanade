-> Matrizes e Vetores
Definição: 
	Uma matriz é uma coleção de variáveis de mesmo tipo, 
acessiveis com um único nome e armazenados contiguamente na memória.
	A individualização de cada variável de um vetor é feita 
através do uso de índices.
	Os vetores são matrizes de uma só dimensão.

Exemplo

caractere vetor[5]; // declara um vetor de 5 posições
caractere matriz[5][3]; // declara uma matriz de 5 linhas e 3 colunas

caractere frutas[4];
frutas[0] = "Maçã";
frutas[1] = "Pera";
frutas[2] = "Uva";
frutas[3] = "Melão";

escreve(frutas[2])

cadeia cesta[][] = {{"Maça", 100}}
